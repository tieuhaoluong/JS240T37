1. Get started

Introduction

Admin 4B is an open source admin template built on top of Bootstrap 4. The source code can be found on the GitHub repo.

Quick start

This template uses many bootstrap features, so you need to have some knowledge in this toolkit. Before you continue, please, take a look at the bootstrap documentation.

For the template work properly you need to add Bootstrap dependencies: jQuery and Popper. Also, add Font Awesome and Simple Line Icons CSS files. Then download the latest release of the template Admin 4B.

Bootstrap files

You do not need to reference bootstrap files because they are already included in admin4b.css and admin4b.js.
